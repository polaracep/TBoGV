# TBoGV - The binding of Gymvod

![art](img/tbogv.png)

Epická úžasná nepřekonatelná hra odehrávající se na gymvodu[^1] (takhle to tam opravdu vypadá!!).

#### Ovládání 
WASD - Movement
E/RMB - Use/Interact
J - Journal
Esc - Pause menu/statistiky


[^1]: Jakákoliv podobnost s reálnými osobami je čistě náhodná
